
# Ledgeway Platform

## What This Is
This repository contains the Dockerised versions of the
Ledgeway platform services, built as part of our Week 5 task.

## Services
Each service is owned by one team member and lives in
its own folder:

- ledgeway-api-gateway
- ledgeway-user-profile-service
- ledgeway-wallet-ledger-service
- ledgeway-notifications-service
- ledgeway-transfer-orchestrator-service
- ledgeway-audit-service

## How We Work
- Each person owns one service
- Each person works on their own branch
- Each person opens a pull request when done
- Do not push directly to main
